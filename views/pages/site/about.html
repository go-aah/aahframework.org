Why aah framework

aah aims to provide flexibility, security, high performance, scalable Web/API framework yet lightweight. aah take cares of infrastructure, boilerplate code and repetitive activities.

aah is not micro/minimalist web framework.

Have a look at features to know your benefits of using aah and its very well documented.

  •   Security aware framework, secure session, CSRF prevention, XSS prevention, authentication, authorization
  •   Provides proven project structure and organization of application pieces.
  •   Extensible at every module/feature. If you're constrained, immediately raise an issue at GitHub.
  •   Maintainable, it grows along with your application.
  •   Gradually maturing and feature sets are getting added.

aah Go web framework initial stable version 0.5 released on May 19, 2017.
