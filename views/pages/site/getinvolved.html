Get Involved

aah framework is an open source project and you can become a part of it. There are various ways you can engage.

Please, let the world know you've contributed to aah framework or give @aahframework a shout out on Twitter or on Google+ to let others know about your contribution.

   Contribute to Documentation

  • You're encouraged to edit, improve and add documentation. For eg: correcting typo mistakes, grammer, add sample code snippet, etc.
  • The docs are all written in GitHub Flavored Markdown.
  • The moments after your edits are merged, they will be automatically published to docs.aahframework.org

   Contribute to Code

  • Test aah, discover and report an issue.
  • Small changes, just send a Pull Request.
  • Bigger changes, I would suggest you to start the discussion first before you work on.
  • I like pull requests that include test cases. If possible example too.
